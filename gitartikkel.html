Git og GitHub


Git er et system for versjonskontroll som kan kjøres i kommandolinjen på din datamaskin.
Git hjelper deg med å holde styr på filene som er lagret et bestemt sted (repository/oppbevaringssted) og forandringer gjort på disse filene. Isteden for å bruke kommandolinjen kan man benytte seg av et program som heter GitKraken. Dette forenkler oppgaven og man slipper å skrive inn koder.

GitHub er et nettsted som tillater deg å laste opp dette oppbevaringsstedet. Det er flere grunner til at man ønsker å gjøre dette; Du har en backup av filene, det gir deg en enklere oversikt enn kommandolinjen og den forenkler samarbeid mellom fler på samme prosjekt.
Ved å bruke GitHub kan flere jobbe med samme prosjekt og man vil enkelt få oversikt over hvem som har gjort hvilke endringer.
Man må altså ikke ha GitHub for å bruke Git, men det gjør bruken lettere.
